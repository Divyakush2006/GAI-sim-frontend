# GovernAI Studio — Frontend

Cinematic simulation interface built with React, Vite, TailwindCSS v4, and Framer Motion.

## Run Locally

**Prerequisites:** Node.js 18+

```bash
npm install
npm run dev
```

The frontend runs on `http://localhost:3000` and proxies API calls to the backend at `http://localhost:8000`.

## Tech Stack

- **React 19** + **Vite 6** — SPA framework
- **TailwindCSS v4** — Utility-first styling
- **Framer Motion** — Animations and transitions
- **Lucide React** — Icon system
